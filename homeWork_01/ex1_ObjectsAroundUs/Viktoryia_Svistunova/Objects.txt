Объект - Домашний питомец.
Состояние :
           Возраст.
           Сытый/Голодный.
           Спит/Бодорствует.
Поведение :
           Стареть.
           Есть.
           Спать.
           Произносить с особой интонацией "мяу" в 3 часа ночи ( либо хочет кушать, либо играть или демоны вселились в пушистого гавнюка).
           
Объект - Телефон.
Состояние :
           Заряд батареи (%).
           Вкл./Выкл.
           Режим звука: вибрация/без звука/звук.           
Поведение :
           Заряжаться/Разряжаться.
           Включаться/выключаться.
           Вибрировать, Издавать звуки (звенеть).
           
Объект - Ноутбук
Состояние :
           Яркость экрана.
           Заряд батареи (%).
           Громкоть звука
Поведение :   
           Увеличивать/уменьшать яркость.
           Заряжаться/Разряжаться.
           Увеличивать/Уменьшать громкоть звука.
           
Объект - Стул
Состояние :
           Устойчивый/Неустойчивый.
           Загрязнения (чистый/грязный).
Поведение :
           Развалиться/Сохранить текущее состояние под влиянием внешнего воздействия.
           Загрязьняться под влиянием внешнего воздействия
          
Объект - Наручные электроные часы 
Состояние :
           Заброкирован/Разблокирован дисплей.
           Заряд батареи (%).
Поведение :
           Не отображать текущее сотсояние даты/времени на экране дисплея (Неактивный дисплей при блокировке экрана).
           Отображает дату, время, количесво шагов и т.д. (Активный дисплей при разблокировке экрана).
           Заряжаться/Разряжаться.

Объект - Верхняя одежда
Состояние : 
           Уровень тепла в верхней одежды (Осенняя/весенняя или зимняя)
           Водонепроницаемость.
Поведение :
           Согревать/не греть в холодную погоду.
           Пропускать/не пропускать влагу в влажную погоду.
           
Объект - Стиральная машина
Состояние :
           Вкл./Выкл.
           Режим стирки.
           Температура стирки.
           Колебания/Покой.
Поведение :
           Стирать в включеном состоянии/стоять не подвижно в выключеном стостоянии.
           Изменять температуру воды при стирке.
           Изменять состояние покоя (вращение барабана) при  стирке.

Объект - Холодильник.
Состояние :
           Вкл. выкл. 
           Температура
Поведение :
           включить - При включении начинает понижаться температура до определенной отметки.
           выключить - При выключении температура начинает повышаться до температуры окружаюещей среды.
           
Объект - Фломастер.
Состояние :
           кол-во чернил.
Поведение :
           писать (при использовании этого поведения объект сам уменьшается кол-во чернил).
     
Объект - Планшет.
Состояние -
           Текущие приложение.
           Уровень звука.
Поведение -
            повысить/понизить уровень звука. 
            перейти на определенное приложение.
